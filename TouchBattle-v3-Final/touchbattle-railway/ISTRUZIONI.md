# TOUCH BATTLE — Deploy su Railway (GRATIS)

## Passi (3 minuti)

### METODO PIÙ SEMPLICE — GitHub + Railway

**Step 1** — Crea repo GitHub
- Vai su https://github.com → "New repository"
- Nome: "touch-battle" → Create
- Carica tutti i file di questa cartella (drag & drop nella pagina del repo)

**Step 2** — Deploy su Railway
- Vai su https://railway.app → "Start a New Project"
- Clicca "Deploy from GitHub repo"
- Seleziona il repo "touch-battle"
- Railway detecta Node.js automaticamente e parte ✅

**Step 3** — Ottieni il link
- Nel progetto Railway → "Settings" → "Networking" → "Generate Domain"
- Ottieni un link tipo: https://touch-battle-production.up.railway.app

**Step 4** — Gioca!
- Manda il link agli amici
- Aprono dal browser del telefono
- Creano/entrano in stanza con codice 4 cifre
- Si gioca in tempo reale!
